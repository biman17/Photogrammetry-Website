ABOUT OUR WEBSITE

The photogrammetric products developed on this website are a Digital Terrain Model (DTM) and Digital Surface Model (DSM) using the process of aerial image matching for the area Boditi, Ethiopia. These products are available in the form of raster representation. Contour lines for each of them are added as vector layers. These products are essential for applications such as urban cadastre, flood modelling, 3D city modelling, and crisis management.

What is a DTM?

DTM can be described as a 3D representation of a terrain surface consisting of X, Y, Z coordinates stored in digital form. It includes not only heights and elevations but other geographical elements and natural features (such as rivers, mountains etc.).


What is a DSM?

An elevation model that includes the tops of buildings, trees, powerlines, and any other objects. Commonly this is seen as a canopy model and only 'sees' ground where there is nothing else covering it.

3D VISUALIZATION

The above maps are further visualized as 3D models of DTM and DSM. Since the study area is a flat land, the variations in the elevation across the layer is not very prominently visible. Qgis2threejs plugin on QGIS software was used to render the 3D views which uses WebGL(Web Graphics Library), a Javascript API for rendering high performance interactive 3D and 2D graphics with any comaptible web browser.
